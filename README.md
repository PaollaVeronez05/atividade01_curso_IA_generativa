
# 💻 Sistema Desenvolvido em Java (IntelliJ IDEA)

## 📌 Sobre o Projeto

Este projeto foi desenvolvido como atividade prática utilizando a linguagem **Java** no ambiente de desenvolvimento **IntelliJ IDEA**.

O objetivo principal foi aplicar conceitos fundamentais de programação, lógica e organização de código, simulando um sistema funcional com base nos requisitos propostos.

---

## 🚀 Tecnologias Utilizadas

*  **Java**
*  **IntelliJ IDEA**
*  Programação Orientada a Objetos (POO)

---

## ⚙️ Funcionalidades

O sistema inclui as seguintes funcionalidades:

*  Cadastro de usuários
*  Validação de login e senha
*  Estrutura de dados utilizando arrays/listas
*  Condições e estruturas de repetição
*  Simulação de acesso ao sistema

---

## 📂 Estrutura do Projeto

```
📁 src
 ┣ 📄 Main.java
 ┣ 📄 Usuario.java (ou equivalente)
 ┗ 📄 Outros arquivos auxiliares
```

---

## 🖥️ Como Executar o Projeto

1. Clone este repositório:

```
git clone https://github.com/seu-usuario/seu-repositorio.git
```

2. Abra o projeto no **IntelliJ IDEA**

3. Localize o arquivo `Main.java`

4. Execute o projeto clicando em ▶️ ou usando:

```
Shift + F10
```
